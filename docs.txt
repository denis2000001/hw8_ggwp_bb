SQL запросы:


CREATE - создаёт новую таблицу данных.

INSERT - добавляет данные в таблицу.

SELECT - получение данных из выбранной таблицы.
         SELECT * FROM <table_name>  -> Этой командой можно вывести все данные из таблицы.

UPDATE - Обновляет данные таблицы.

DROP - удаляет строки(ку) из таблицы, всю таблицу целиком или целую базу данных.
